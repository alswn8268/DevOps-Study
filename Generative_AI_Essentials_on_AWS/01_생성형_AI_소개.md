# 생성형 AI 설명
> 방대한 훈련 데이터를 사용하여 패턴과 관계를 학습하며, 이를 통해 학습된 구조를 반영하는 새로운 콘텐츠를 만듦

### 고려해야 할 당면 과제
- 할루시네이션
- 지식 재산권 문제
- 데이터 보안 및 개인 정보 보호
- 편향 및 공정성
- 해석 가능성
- 윤리 및 의사 결정

# 파운데이션 모델
> 전이 학습을 위한 다양하고 강력한 기반을 마련하는 모델
## 파운데이션 모델 적용
> 파운데이션 모델은 다용도성을 지니며 다양한 태스크를 수헹할 수 있음
- 텍스트 생성
- 텍스트 요약
- 정보 추출
...etc

### 파운데이션 모델을 조정하려면 다음 2단계를 거쳐야 함
1. 사전 훈련: 다양한 데이터 세트에 대한 초기 훈련
2. 조정: 특정 태스크나 영역에 맞게 모델을 파인 튜닝 또는 프롬프팅

## 모델 학습 방법
> 훈련 데이터 -> ML 알고리즘 -> 모델
### 대규모 언어 모델(LLM)
- 방대한 텍스트 데이터를 기반으로 훈련되는 고급 AI 시스템
- 변환기와 같은 복잡한 신경 아키텍처를 사용하여 사람이 쓴 것과 유사한 텍스트를 이해하고 생성
### 토큰
- LLM이 처리하는 기본 단위(단어, 형태소 또는 문자 등)
### 임베딩
- 고차원 공간의 토큰을 수치로 나타내며, 각 토큰에는 의미와 관계를 나타내는 벡터가 할당됨
=> 의미론적 유사성 이해 가능

### 훈련 데이터
> 훈련 과정에서 AI에 입력되는 대량의 정보
- 품질이 높고 다양한 데이터가 필요
- 정형, 비정형 데이터를 모두 사용하나 비정형 데이터는 처리 및 분석을 위한 고급 기술이 필요한 경우가 많음

### ML 알고리즘
> AI 시스템의 두뇌 또는 학습 방법
- 패턴을 인식하고 이러한 패턴을 사용하여 새롭고 유사한 콘텐츠를 생성하는 방법을 학습

### 모델
> 기계 학습 알고리즘을 훈련 데이터에 적용한 결과
- 모델의 품질은 사용된 훈련 데이터와 ML 알고리즘의 질에 따라 결정됨
- 모델 개발은 반복적인 경우가 많음
