# 12.1 gzip 명령어를 이용한 압축 및 해제
## 파일 압축

> gzip [-v] filename(s)

### gzip 압축 파일 생성
```
gzip etc.tar # etc.tar.gz 파일 생성
```

### 파일 정보 확인
> gzcat filename
```
[vagrant@user01 ~]$ file etc.tar.gz
etc.tar.gz: gzip compressed data, was "etc.tar", last modified: Tue Oct 14 10:52:11 2025, from Unix, original size modulo 2^32 22609920
```

### 압축 해제 후 원본 삭제
```
[vagrant@user01 ~]$ gunzip etc.tar.gz
```

### 압축된 tar 파일의 내부 내용 확인
```
[vagrant@user01 ~]$ tar -ztvf etc.tar.gz
```
