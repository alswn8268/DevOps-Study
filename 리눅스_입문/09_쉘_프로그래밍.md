# 9.1 쉘 프로그래밍 개요
## sh 명령어 사용
```
[vagrant@user01 ~]$ vi sample.sh

#!/bin/bash
echo "Hello, World!"

# 실행하고자 하는 스크립트 파일명 앞에 sh를 붙여서 실행. 실행권한 없어도 가능
[vagrant@user01 ~]$ sh sample.sh 
Hello, World!
```
* `셔뱅(#!)`: 무엇으로 실행할 지 알려주는 역할
## chmod 명령어 사용
```
[vagrant@user01 ~]$ chmod +x sample.sh # chmod로 실행 권한 주면 됨
[vagrant@user01 ~]$ ./sample.sh
Hello, World!
```
