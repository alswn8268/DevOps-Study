## AWS 요금
### AWS 프리 티어
> AWS 프리 티어를 이용하면 지정된 기간 동안 비용을 들이지 않고 특정 서비스를 사용할 수 있음
- `상시 무료`: 만료 기간이 없고 모든 AWS 고객이 이용할 수 있음. (ex. Lambda, DynamoDB)
- `12개월 무료`: AWS에 처음 가입한 날로부터 12개월 동안 무료 (ex. 일정량의 S3 Standard 스토리지, 월별 EC2 컴퓨팅 시간 한도 등)
- `평가판`: 서비스를 활성화한 날짜부터 무료 평가판 제공 (ex. Inspector 등)

### AWS 요금 개념
- `종량제`: 사전에 용량을 프로비저닝하지 않고 사용한 리소스에 대해서만 비용 지불
- `예약`: 용량을 예약하여 비용 절감
- `볼륨 기반 할인`: 사용량이 증가함에 따라 볼륨 기반 할인을 통해 비용 절감

## 통합 결제
- AWS Organizaions의 통합 결제 기능을 사용하면 조직의 모든 AWS 계정에 대한 단일 청구서를 수령
- 최대 4개 (+추가 문의) 계정이며, 모든 연결 계정의 결합 비용 추적 가능
- 대량 구매 할인

## AWS 요금 관련 도구
### AWS Budgets
- AWS 서비스 사용량 및 비용에 대한 임계값을 설정하는 데 사용할 수 있는 도구
### AWS Cost Explorer
- 시간 경과에 따른 AWS 비용 및 사용량을 시각화하고 이해하고 관리하는데 사용할 수 있는 도구

## AWS Support 플랜
- 기술 문서, 설명서 및 지원 커뮤니티에 대한 엑세스가 모든 AWS 고객에게 무료로 제공됨
- Developer / Businees / Enterprise On-Ramp / Enterprise
### Technical Account Manager (TAM)
- 지침, 기술 전문 지식 및 모범 사례를 제공하는 AWS의 주 연락창구
- Enterprise On-Ramp, Enterprise Support 플랜만 포함

## AWS Marketplace
- AWS에서 실행되는 서드 파티 소프트웨어 리스팅을 제공하는 디지털 카탈로그
- 인프라 제품, 비즈니스 애플리케이션, 데이터 제품 및 DevOps 등 여러 범주의 제품을 제공하며, 각 범주에서 하위 범주의 상품 리스팅을 탐색할 수 있음
