# 3.1 디렉토리 작업
## 현재 작업 디렉토리 확인
```pwd```
## 디렉토리 내용 확인
```ls options pathname```
## 숨겨진 파일 보기
```ls -a```
## 디렉토리 내용 자세히 보기
```ls -l```
## 개별 디렉토리 확인
```ls -d```
## 디렉토리 하위 목록 보기
```ls -R```
## 파일 종류 확인
- ```ls -F```: 각 명령어 결과와 차이점
- ```ls -l```: 명령어의 가장 첫 필드가 파일의 종류
- ```file```: 특정 파일이 어떤 종류의 파일인지 확인
